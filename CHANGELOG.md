# Changelog - CASHIER WATCH

## [2.4.0] - 2026-03-19
### Added
- **Mobile Responsiveness**: Полная адаптация интерфейса под мобильные устройства.
- **Bottom Navigation**: Удобное меню для переключения вкладок на смартфонах.
- **Incident Details**: Расширенный просмотр информации об инцидентах.
- **Process Logic Tab**: Новый компонент `ProcessGraphViewer` для визуализации и настройки цепочек операций (SVG-граф).
- **System Metadata**: Обновлены системные файлы проекта (package.json, metadata.json).

### Changed
- Оптимизирована верстка таблиц инцидентов и моделей для маленьких экранов.
- Улучшена визуализация алертов в режиме реального времени.

### Fixed
- Исправлено перекрытие элементов в панели мониторинга на узких экранах.
