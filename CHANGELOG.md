# Changelog - CASHIER WATCH

## [2.5.0] - 2026-03-28
### Added
- **Real-time WebSocket Integration**: Полноценная поддержка WebSockets для мгновенной доставки инцидентов, системных метрик и данных о производительности моделей.
- **Workflow Compliance Engine**: Интеграция логики проверки регламента кассовых операций (на базе `workflow_config.json`).
- **Enhanced Incident Cards**: Новый дизайн карточек инцидентов с поддержкой медиа-превью, тегов объектов и расширенной фильтрации (Severity, Status, Type).
- **System Monitoring (Tech Tab)**: Живые графики нагрузки GPU, температуры, использования VRAM и сетевого трафика.
- **Model Performance Monitoring**: Отображение FPS и задержки инференса в реальном времени для активных моделей.
- **Logging Specification (Task 4.4)**: Создан документ `logging_spec.md`, определяющий 3 уровня логирования и ротацию данных.
- **Backend Guide & Integration TS**: Добавлены инструкции для разработчиков бэкенда и спецификация для интеграции с Google Colab.

### Changed
- **Header Stability**: Исправлены скачки верстки в шапке при изменении статуса WebSocket соединения.
- **Process Graph**: Обновлена визуализация графа процессов для соответствия актуальному регламенту валютообменных операций.
- **API Architecture**: Переход на FastAPI для бэкенда с поддержкой асинхронных WebSocket-трансляций.

### Fixed
- Исправлены ошибки типизации при обработке входящих сообщений WebSocket.
- Улучшена обработка ошибок при отсутствии подключения к бэкенду (fallback на mock-данные).

## [2.4.0] - 2026-03-19
### Added
- **Mobile Responsiveness**: Полная адаптация интерфейса под мобильные устройства.
- **Bottom Navigation**: Удобное меню для переключения вкладок на смартфонах.
- **Incident Details**: Расширенный просмотр информации об инцидентах.
- **Process Logic Tab**: Новый компонент `ProcessGraphViewer` для визуализации и настройки цепочек операций (SVG-граф).
- **System Metadata**: Обновлены системные файлы проекта (package.json, metadata.json).

### Changed
- Оптимизирована верстка таблиц инцидентов и моделей для маленьких экранов.
- Улучшена визуализация алертов в режиме реального времени.

### Fixed
- Исправлено перекрытие элементов в панели мониторинга на узких экранах.
