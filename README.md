# CASHIER WATCH v2.4 — Интеллектуальная видеоаналитика кассовой зоны

**CASHIER WATCH** — это прототип системы контроля кассовых операций, предназначенный для выявления нарушений и ошибок сканирования в режиме реального времени.

## 🚀 Быстрый запуск

### 1. Основной интерфейс (React + Vite)
Самый мощный и детализированный интерфейс для продакшена.
```bash
# Установка зависимостей
npm install

# Запуск в режиме разработки
npm run dev
```
После запуска откройте `http://localhost:3000`.

### 2. Python-интерфейсы (Streamlit / Gradio)
Альтернативные варианты для быстрой итерации и ML-экспериментов.
```bash
# Установка Python зависимостей
pip install -r requirements.txt

# Запуск Streamlit (Аналитическая панель)
streamlit run app.py

# Запуск Gradio (ML-демо)
python gradio_app.py
```

## 🛠 Архитектура и Безопасность
Система спроектирована для работы в **закрытом контуре (Air-Gap)** на локальных серверах заказчика:
*   **On-Premise**: Все данные остаются внутри локальной сети.
*   **Edge Computing**: Обработка видео происходит на локальном GPU (NVIDIA RTX).
*   **Mode Switching**: Поддержка режимов **DEMO** (архив) и **REALTIME** (живой поток).

## 📦 Функциональные модули
1.  **Мониторинг СБ**: Видеоплеер с AI-метаданными и мгновенными алертами.
2.  **Журнал инцидентов**: База данных нарушений с фильтрацией и детальными карточками.
3.  **MLOps Панель**: Управление версиями моделей и мониторинг системного стека.
4.  **Техническая панель**: Мониторинг "здоровья" сервера (CPU/GPU, Disk, Network).
